The maze object must be instantiated with the size parameter, which will be used to
create the array that the algorithm will use to navigate to the center. For now, this
will only work with square mazes.


An example 8x8:
```
6 5 4 3 3 4 5 6
5 4 3 2 2 3 4 5
4 3 2 1 1 2 3 4
3 2 1 0 0 1 2 3
3 2 1 0 0 1 2 3
4 3 2 1 1 2 3 4
5 4 3 2 2 3 4 5
6 5 4 3 3 4 5 6
```
